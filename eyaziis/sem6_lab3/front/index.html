Предложение

Результаты семантико-синтаксического анализа

Используйте кнопку "Выберите файл", чтобы выбрать файл, содержащий анализируемое предложение. После выбора файла семантико-синтаксический анализ начнётся автоматически.

По завершению анализа в разделе "Результаты семантико-синтаксического анализа" отобразится таблица, в которой будут зафиксированы все лексемы проанализированного предложения, и все возможные значения семантико-синтаксических функций для этих лексем.

Нажатие левой кнопкой мыши по любой ячейке таблицы (кроме заголовочных) откроет окно редактирования значения ячейки.

После успешного завершения семантико-синтаксического анализа отобразится кнопка "Сохранить". Нажатие на кнопку "Сохранить" сохранит таблицу в её текущем виде (с учётом Ваших изменений) в JSON-формате в файл с названием "decomposition.txt".
