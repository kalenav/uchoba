Редактирование

Нажмите кнопку "Выберите файл", чтобы выбрать файл, содержащий анализируемое предложение. Синтаксический анализ предложения начнётся автоматически.

По завершению синтаксического анализа прочитанное из файла предложение будет отображено в разделе "Предложение", а результат его синтаксического анализа согласно грамматике - в разделе "Разобранное предложение". Также по завершению синтаксического анализа станут доступны кнопки "Сохранить" и "Редактировать".

Нажатие на кнопку "Сохранить" сохранит текущее содержимое раздела "Разобранное предложение" в файл с названием decomposition.txt.

Нажатие на кнопку "Редактировать" откроет окно редактирования содержимого раздела "Разобранное предложение". После внесения желаемых изменений нажмите кнопку "Подтвердить", чтобы перезаписать содержимое раздела "Разобранное предложение". Нажмите кнопку "Отмена", чтобы отменить редактирование.
